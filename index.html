HOW IT WORKS
📱
HOST CREATES A ROOM
One person hosts. Everyone else scans the QR code or enters the room code to join.
👆
HOST READS THE QUESTION
Host taps the box to arm the buzzers. Players race to buzz first!
🏆
FIRST BUZZ WINS THE TURN
The fastest player's name appears. Host taps ✓ RIGHT or ✕ WRONG.
⚠️
WRONG? PENALTY MODE
1st wrong → team locked out for 10s, others get a chance. 2nd wrong → open to everyone.
🔄
OR USE TIMER MODE
No penalties. Host controls a visible countdown timer. Players buzz freely. Simple and fast.
🎨
TEAMS · THEMES · QR SHARING
Create teams, pick a theme, share via QR. Works on any phone — no app needed.
